{ 以下哪種情況最符合您的要求呢 ? }

  • 我只想專心寫程式碼

    MIT License 是一個寬鬆的授權， 它允許別人用你的程式碼做任何事情，但必須保證你的所有權，並且你無須承擔程式碼使用產生的風險。

    {% include using-sentence.html license-id="mit" %}

  • 我想保護我的專利

    Apache License 2.0 與 MIT License 比較類似, 但提供了專利貢獻者的明確授權，使用者需要放置版權說明。

    {% include using-sentence.html license-id="apache-2.0" %}

  • 我關心專案分享與改進

    GNU GPLv3 是一個 Copyleft 授權， 所有使用了你源碼的人都必須按照相同的授權提供源碼， 使用者需要放置版權說明。

    {% include using-sentence.html license-id="gpl-3.0" %}
